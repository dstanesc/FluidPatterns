# FluidFramework - Fluid Patterns

Main focus is investigating the experimental Property DDS and related FluidFramework software patterns


# Getting Started

Needed dependencies
```
$ node --version
v16.13.1

$ npm --version
8.1.2
```

```
npm run clean
npm install --legacy-peer-deps
npm run build
```
